# Frontend Developer Job Description Summary

## Overview
This role is for an early-career Frontend Engineer (0-2 years experience) to build and maintain user-facing web applications with a focus on performance, accessibility, and clean code.

---

## Key Responsibilities

- **Development**: Build responsive web UIs using **React.js**, JavaScript (ES6+), HTML5, and CSS3/SCSS
- **Design Collaboration**: Translate Figma/design mockups into pixel-perfect, production-ready components
- **State Management**: Implement Redux, Zustand, or Context API for efficient data flow
- **API Integration**: Work with backend engineers to integrate RESTful APIs, GraphQL, and WebSockets
- **Code Quality**: Write clean, testable, well-documented code; participate in code reviews
- **Performance**: Optimize components through lazy loading, code splitting, and memoization
- **Testing**: Write unit tests (Jest, React Testing Library) covering happy paths and edge cases
- **Debugging**: Troubleshoot frontend performance, security issues, and cross-browser compatibility
- **Collaboration**: Work in agile sprints with designers, product teams, and engineers

---

## Requirements

### Experience
- 1–3 years of frontend development experience (including internships, academic, or personal projects)
- Professional experience with React.js and its ecosystem

### Technical Skills
- **Core**: JavaScript (ES6+), TypeScript, HTML5, CSS3, SCSS, responsive design
- **Framework**: React.js (hooks, functional components, lifecycle), React Native (nice to have)
- **State Management**: Redux, Zustand, or Context API
- **Build Tools**: Webpack, Babel, Vite, or Parcel
- **API Handling**: RESTful APIs, async patterns (fetch, axios, async/await)
- **Testing**: Jest, React Testing Library (nice to have)
- **Version Control**: Git/GitHub
- **UI Libraries**: Material-UI, Tailwind CSS, or Ant Design

### Soft Skills
- Strong problem-solving and debugging abilities
- Clear communication in collaborative team environments
- Growth mindset and eagerness to learn
- Ability to work independently and in fast-paced agile settings

---

## Nice to Have
- TypeScript experience
- App deployment (Google Play Store/Apple App Store)
- Consumer-facing product experience
- Performance optimization techniques (lazy loading, memoization, bundle splitting)
